# DUBOIS (Data Science for everyone)

Welcome! This repository contains Data 6 materials presented at the **DUBOIS Symposium on Undergraduate Data Science Education** hosted by UC Berkeley's College of Computing, Data Science, and Society.
These resources are part of the Data 6 curriculum and are designed to support instructors, educators, and workshop participants interested in introductory data science education.

## DUBOIS Website: https://dubois-ctds.github.io/
